 **사용자가 직접 입력한 음표 조합**으로 **8비트 음악을 생성**

---

## 사용 방법 예시


```
도미솔도+
```

```
도 레 미 파 솔 라 시 도+
```

```
도 레 미 레 도 레 미 솔+
```

* \*\*도+\*\*는 높은 도 (C5)
* **띄어쓰기**는 무시되며, `쉼표` 역할로 `" "`를 쓸 수 있음

---

## 🎧 결과

* 입력한 음 순서대로 8비트 스타일로 재생
* 사각파 기반이므로 클래식한 게임 느낌
* 간단한 테마송이나 알림음 작곡에 적합
